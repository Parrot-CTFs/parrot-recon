# Parrots Recon ( A Recon Automation Script )


install:
```
git clone https://github.com/parrotassassin15/parrot-recon.git
```

```
sudo chmod +x install.sh ; sudo ./install.sh
```

run: 
```
sudo ./parrot-recon.sh <domain>
```
